# CloseRequested

[common]\
data object [CloseRequested](index.md) : [WindowEvent](../index.md)

L'utilisateur a demandé la fermeture de la fenêtre (bouton ×, Alt+F4, ⌘W, etc.).

L'application reste libre d'ignorer ou de différer la fermeture.
